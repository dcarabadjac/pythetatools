# pythetatools
The python package providing post-fit analysis and drawing tools for P-theta analysis outputs.

To install the package run:
pip install -e "/path/pythetatools/"

The documentation is automatically generated by Sphinx. It can be found here:
pythetatools/docs/_build/html.index.html

To update the documentation run from docs folder:
sphinx-apidoc -o . --separate ../pythetatools
